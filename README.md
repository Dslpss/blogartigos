# BlogArtigos

Portal de notícias e artigos premium com visual moderno de agência high-end.

## Tecnologias
- **Framework**: Next.js
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Como rodar
1. Instale as dependências: `npm install`
2. Rode o servidor de dev: `npm run dev`
